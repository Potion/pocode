Global Mirror

One of the capabilities defined is the Mirror capability. This capability allows mirroring of the output data. This is useful when the sensor is placed in front of the user (for example, by the TV), and the user sees itself from behind in the application.

In most cases, once a specific node is mirrored, the application would probably want all nodes to be mirrored. OpenNI supplies a mechanism called Global Mirror, which, when turned on, changes the mirror state of every node that supports this capability. It will also set the mirror state of new nodes, once created.
